GETTING STARTED

Welcome to the Halvor Lines Speedway Marketing Tool

This tool allows you to create professional marketing materials for Halvor Lines Speedway events using pre-designed templates. Here's how to get started:

  1. Browse Templates: Start by exploring the available templates on the home page. Templates are organized by category (Race Results, Bus Races, Silver 1000, etc.).
  2. Select a Template: Click on a template that matches your needs to view details, then click "Customize" to open it in the editor.
  3. Customize Your Design: Use the editor tools to modify text, colors, images, and layout to fit your specific event.
  4. Export Your Design: When you're satisfied with your design, export it in your preferred format (PNG, JPG, PDF, or SVG).
  5. Share Your Creation: Use your exported design on social media, in emails, or on the Halvor Lines Speedway website.

System Requirements

The Halvor Lines Speedway Marketing Tool works best with:

  • Browsers: Chrome, Firefox, Safari, or Edge (latest versions)
  • Devices: Desktop, laptop, or tablet (mobile phones supported but limited functionality)
  • Internet: Stable internet connection required
  • Screen Resolution: Minimum 1024x768 recommended

TEMPLATE BROWSER

Finding the Right Template

The template browser is your starting point for creating marketing materials:

  • Categories: Templates are organized by category in the sidebar. Click a category to filter templates.
  • Preview: Each template card shows a preview image and description of the template's purpose.
  • Platform: Templates are designed for specific platforms (Facebook, Instagram, Twitter, Email, Website).
  • Customize: Click the "Customize" button on any template to open it in the editor.

Template Categories

Available template categories include:

  • Race Results: Templates for sharing race outcomes on various platforms
  • Bus Races: Templates for promoting the Bus Races special event
  • Silver 1000: Templates for promoting the Silver 1000 championship event
  • Email Templates: Templates for email marketing campaigns
  • Website Content: Templates for website banners and content

TEMPLATE EDITOR

Editor Overview

The template editor is where you'll customize your selected template:

  • Canvas: The main editing area where you can see your design
  • Control Panel: The sidebar containing editing tools and options
  • Template Options: Customize backgrounds, layouts, and color themes
  • Text Controls: Edit text content, fonts, sizes, and styles
  • Image Controls: Upload and place images in your design
  • Export Options: Save your design in various formats

Working with the Canvas

Tips for working with the canvas:

  • Select Elements: Click on any element in the canvas to select it for editing
  • Move Elements: Click and drag selected elements to reposition them
  • Resize Elements: Use the corner handles to resize selected elements
  • Delete Elements: Select an element and press Delete or Backspace to remove it
  • Preview: Use the Preview button to see how your design will look when exported

IMAGE UPLOAD

Adding Images to Your Design

You can add your own images to customize your designs:

  1. Find the Image Section: In the editor control panel, locate the Images section
  2. Upload Image: Click the upload button or area to select an image from your device
  3. Select Image: Choose an image file (JPG, PNG, or GIF recommended)
  4. Place Image: The image will be added to your canvas where you can position it
  5. Adjust Image: Resize, rotate, or reposition the image as needed

Image Tips

For best results with images:

  • Image Size: Use high-resolution images (1200x1200px or larger recommended)
  • File Format: PNG for graphics with transparency, JPG for photos
  • File Size: Keep files under 5MB for optimal performance
  • Aspect Ratio: Match the aspect ratio to the template when possible
  • Image Library: Previously uploaded images are saved in your session for reuse

TEXT EDITING

Editing Text Elements

To edit text in your design:

  1. Select Text: Click on any text element in the canvas to select it
  2. Edit Content: Use the text input field in the control panel to change the text
  3. Change Font: Select a different font from the Font Family dropdown
  4. Adjust Size: Use the Font Size slider to make text larger or smaller
  5. Apply Styles: Use the Bold, Italic, and Underline buttons to style your text
  6. Change Color: Select a color from the color palette or use the color picker

Text Tips

For effective text in your designs:

  • Keep it Concise: Use short, impactful messages
  • Font Choice: Use easy-to-read fonts. Racing Sans One and Oswald are recommended.
  • Hierarchy: Use different font sizes and weights to create a clear visual hierarchy
  • Contrast: Ensure good contrast between text color and background color
  • Proofread: Always double-check your text for typos and errors

EXPORTING DESIGNS

Saving Your Work

Once your design is complete, you can export it in several formats:

  • PNG: Best for web graphics with transparency
  • JPG: Best for photos and web graphics without transparency
  • PDF: Best for print or sharing documents
  • SVG: Scalable vector format for high-resolution needs

To export, click the "Download Image" or "Download PDF" button in the editor. You can also copy the design to your clipboard for pasting into other applications.

FREQUENTLY ASKED QUESTIONS

Q: Can I use my own fonts?

A: Currently, the tool supports a curated list of fonts optimized for racing promotions. We may add custom font uploads in the future.

Q: What if I accidentally close the editor?

A: The editor has an autosave feature that saves your work periodically. You should be able to resume your last session if you reopen the template.

Q: Can I collaborate with others on a design?

A: The current version of the tool is designed for single-user editing. Collaboration features may be considered for future updates.

Q: Who do I contact for support?

A: For technical support or questions, please contact the Manus team.
